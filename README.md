# 黒い砂漠 成長日記帳 Pro (Dark Mode)

『黒い砂漠』の戦闘・生活ステータス、および装備構成を日誌形式で保存・管理できるデスクトップアプリケーションです。
AI（Gemini）とを使うことで、ゲーム内のステータス画面のスクリーンショットからテキストを自動抽出し、手入力をほぼゼロにして記録をつけることができます。

さらに、蓄積されたデータから**レベルアップまでにかかった日数**や**装備の更新履歴**を自動計算し、グラフで視覚化するダッシュボード機能も搭載しています。

---

## 📊 主な機能

* **戦闘・生活ステータスの管理**: ゲーム内の並び順に準拠したUIで、レベルや経験値を記録。
* **装備構成の記録**: 15箇所の装備スロットの状態を日付ごとに保存。
* **AI連携自動入力**: 画面左上のボタンからプロンプトをコピーし、Geminiにスクリーンショット（ゲーム内Pで表示できるウィンドウの生活タブ＋左上の戦闘経験値をWin＋Alt＋Sで部分スクショしたもの or Pの生活タブを出した全体のスクショでも可能なはず...）を読ませて出力されたJSONを貼り付けるだけで、一瞬でフォームに入力されます。
* **ドラッグ＆ドロップ対応**: Geminiから出力された `.json` ファイルをそのままアプリにドロップして読み込ませることも可能です。
* **成長ダッシュボード (HTML自動生成)**:
    * 次のレベルに上がるまでに「何日かかったか」の自動計算
    * 「どの部位の装備がいつ更新されたか」の履歴表示
    * 生活等級のランク（初級〜道人）に応じたゲーム内イメージカラーでの成長折れ線グラフ表示

---
